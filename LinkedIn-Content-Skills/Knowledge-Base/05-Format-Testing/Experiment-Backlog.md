## Experiment Template

```markdown
### Experiment: [Title]

**Hypothesis:** [What we think will happen]
**Author:** [Who posts]
**Format:** [Text / Carousel / Video / Multi-image / etc.]
**Topic:** [What the post is about]
**Control:** [What we're comparing against — previous post / average / specific post]
**Metric:** [What we measure — impressions, engagement rate, saves, comments]
**Duration:** [How long to wait for results — typically 7 days]

**Result:** [Fill after experiment]
**Learning:** [What we now know]
**Action:** [Update DIP / change strategy / test further]
```
