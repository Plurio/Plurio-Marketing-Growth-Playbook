## DECLINING (stop using)

| Trend | Why Declining | Replace With |
|-------|-------------|-------------|
| **"Swipe carousel" generic tips** | LinkedIn targeting low-value carousels. Over-saturation. Carousel fatigue | Carousels with ORIGINAL data/frameworks only. 6-9 slides max (not 12+) |
| **Prompt-sharing posts** ("I built a super prompt for X") | Format fatigue. People tired of "comment PROMPT for free" | Show the RESULT of the prompt, not the prompt itself |
| **"5 agents every founder needs"** listicles | Overused format in AI niche. Declining engagement | Share ONE specific agent use case with real numbers |
| **Engagement-focused polls** | -35% reach. Used as engagement hack, not genuine research | Only for real audience research with follow-up post |
| **"Link in first comment"** | LinkedIn now penalizes this pattern (2026). Algorithm identifies bridge behavior | Provide full value in post. If link necessary, own it in body |
| **Hashtag stuffing (5+)** | 360Brew doesn't need hashtags for topic classification. >3 = spam risk | 1-3 relevant hashtags max. Natural keywords in text matter more |
| **Daily posting** | 2+ posts/day = -40% reach per post. Algorithm penalizes quantity | 3-5 posts/week max. Quality > frequency |

## KEY ALGORITHM SHIFT: What This Means for Content Strategy

### 360Brew = Semantic Reasoning (not signals)

The algorithm now **reads and understands content** like a human, rather than summing numeric features. That changes everything:

| Before (signal-based) | Now (360Brew semantic) |
|----------------------|--------------------------|
| Hashtags = discovery lever | Hashtags = minor signal (1-3 max) |
| Formatting tricks boost reach | Formatting helps AI parse, but substance matters more |
| Post fast, post often | Post deep, post quality (3-5/week) |
| Engagement pods for boost | Pods = shadow ban |
| Viral hooks > substance | First sentence = semantic anchor for entire post ranking |
| Network size = reach | Topic expertise = reach (60% goes to interest-based clusters) |
| Any engagement = good | Only thoughtful engagement counts (saves, deep comments, shares) |

### Practical Implications for Our Posts

1. **Profile-Content Alignment is mandatory.** Headline + About + Experience must match post topics. ~90 days to build semantic authority.
2. **The first sentence = the anchor.** 360Brew "Lost-in-Distance" effect: a generic prologue → deprioritization.
3. **Saves > everything else.** Create save-worthy content: frameworks, checklists, reference material.
4. **Links = -60% reach.** Avoid as much as possible. Value in the post, not behind a link.
5. **Niche depth = advantage.** 360Brew zero-shot reasoning is excellent at matching niche authors with targeted audiences.
