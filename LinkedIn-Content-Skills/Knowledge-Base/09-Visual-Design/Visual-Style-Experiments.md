# Visual Style Experiments — Lab Journal (Template)

> Journal of experiments with AI-generated visuals for LinkedIn posts. Goal — after 5–10 cases, establish a stable visual language for each author and cluster that will go into the DIP and into `Visual-Brief-Engine.md` as defaults.
>
> **When to log:** on every visual generation (Higgsfield / HeyGen / Lovable / Midjourney / any image model). After 5–10 entries, stable patterns emerge.

---

## How to use this journal

After each generation — add an entry to the **Experiments** section. Fields:

- **Date** — generation date (not publication date)
- **Post** — post file slug
- **Author** — who publishes
- **Cluster** — which content cluster (Founder Journey / Performance Marketing / AI Workflow / ...)
- **Mode** — which Visual mode was chosen (Hero-Photo / Builder / Data-viz / Conceptual / ...)
- **Anchor** — what served as the visual anchor (metaphor / scene / object)
- **Prompt** — full prompt for the model
- **Model** — Higgsfield / HeyGen / Midjourney / Stable Diffusion / ...
- **Generations** — how many variants were generated
- **Pick** — which variant was chosen
- **URL** — link to the final asset (if stored in the cloud)
- **Reaction** (fill after publication) — engagement, individual comments on the visual
- **Lesson** — what we learned

### Universal lessons

- **Anchor strength:** a concrete object / scene > an abstract metaphor.
- **Aspect ratio:** Twitter cross-post = 16:9 or 1:1; LinkedIn = 4:5 or 1:1.
- **AI artifacts:** check hands, in-frame text, reflections.
- **Brand consistency:** one color palette / style across 3+ posts in a row = increases recognizability.

---

## Anti-patterns

- Generic stock-photo vibe ("business people shaking hands")
- Visual repeats the post text literally (illustrate the pitch)
- AI-faces without a uncanny-valley check
- A visual that makes the text redundant — it should ADD a layer, not duplicate
