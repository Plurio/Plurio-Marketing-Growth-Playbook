# Digital Identity Profile — Portable Template

> Template for creating a DIP for any author and any platform.
> Filled from RAW SOURCES: transcripts, voice input, the author's personal posts.
> NEVER from AI-generated content.

---

## Filling Instructions

### Where to Source Data (Canonical Sources)

| Priority | Source Type | Example |
|----------|-------------|---------|
| Tier 1 | Public appearances | Podcasts, conferences, webinars |
| Tier 2 | Work discussions | Client calls, strategy meetings |
| Tier 3 | Personal notes | Telegram, voice memos, notes, CV |

### How to Fill Tone of Voice

1. Listen to / read 3-5 of the author's transcripts
2. Write down characteristic phrases VERBATIM
3. Pay attention to: sentence length, repeating constructions, transitions between thoughts
4. Capture what the author NEVER says

### Minimal DIP (v0.5) — 1-2 sources

Fill sections 1, 2, 3. The rest — as more sources become available.

### Production DIP (v1.0) — 5+ sources

All 9 sections complete, Stories Bank contains 10+ stories, Feedback Log maintained after every publication.

- [ ] All data from raw sources (not from posts/copy)
- [ ] Specific sources listed (transcript files)
